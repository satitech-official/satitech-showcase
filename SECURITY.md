# Security Policy

## Reporting a vulnerability

Please do not open a public issue for a security vulnerability. Email **satitechinfo@gmail.com** with:

- the affected component;
- steps to reproduce the issue;
- the potential impact; and
- any suggested mitigation.

SatiTech will acknowledge a complete report as soon as reasonably possible and coordinate a responsible fix. Never include credentials, personal data or destructive proof-of-concept material in a report.

Only the latest version on the `main` branch is supported.
